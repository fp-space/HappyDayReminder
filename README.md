一个简单实用的生日提醒工具,可以自动发送生日提醒邮件,支持公历和农历生日,可自定义提醒模板

## ✨ 特性

- [X] 支持公历和农历生日提醒
- [X] SMTP邮件发送
- [X] 自定义 Handlebars 邮件模板
- [X] 支持在配置文件批量管理多个生日提醒
- [X] 可预览渲染后的电子邮件
